#正则 #编程 #字符串

```python
>>> 'a b   c'.split(' ')
>>> ['a','b','','','c']

>>> re.split(r'\s+','a b   c')
>>> ['a','b','c']
>>> re.split(r'[\s\,]+','a,b,c,d')
>>> ['a','b','c','d']
```

## 分组

用 `()`表示 提取的分组(Group)

```python
>>> m = re.match(r'^(\d{3})-(\d{3,8})$)','010-12345')
>>> m.group(0)
>>> '010-12345'
>>> m.group(1)
>>> '010'
>>> m.group(2)
>>> '12345'
```
